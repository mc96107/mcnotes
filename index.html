About

Organized, note taking app with wiki connections and markdown syntax. Syncs with dropbox and remotestorage.

Tips

  • Edit using markdown
  • To rename a note, open it and click on the title.
  • To add a folder, create a note under it: folder/note.md
  • Link between notes by using "[[" eg: [[bookmarks]] (without adding the extension .md)
  • On a FirefoxOS device, when sharing a link with the app, it creates a note named bookmarks.md (if it noes not exist) and adds the url.
